## What is this?
Example on how use React DnD.

## Installation

```bash
git clone https://github.com/andresgutgon/donald-trash.git
cd donald-trash
npm install
npm run start
open http://localhost:3000
```

## Deploy

```bash
npm run deploy
```

## Run tests

```bash
npm run test
```
